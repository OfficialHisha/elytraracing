# Elytra Racing Wiki

Welcome to the Elytra Racing documentation! This plugin allows you to create and manage high-speed aerial races on your Minecraft server.

## Key Features

*   **Custom Race Tracks:** Use the Elytra Racing Tool to place rings and define race borders with ease.
*   **Dynamic Scoreboards:** Real-time tracking of race time, lap counts, and player rankings.
*   **Special Rings:** Configure optional rings that execute console commands (e.g., speed boosts, firework effects) when passed.
*   **Race Management:** Comprehensive control over laps, countdowns, DNF timers, and automated resets.
*   **Spectator Mode:** Admins can join as spectators to monitor races with dedicated teleportation tools.
*   **PlaceholderAPI Support:** Display player stats like fastest lap times in other plugins.
