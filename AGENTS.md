This is a Minecraft plugin that implements an elytra racing game.

## For AI Agents

When working on this repository, please follow these guidelines:

*   This project uses the PaperMC API.
*   The project is built using Maven.
*   The main source code is located in `src/main/java`.
*   Tests are located in `src/test/java`.
*   The plugin is built to `target/elytraracing-0.1.0-SNAPSHOT.jar`.
*   The plugin is copied to `server/plugins` after a successful build.
